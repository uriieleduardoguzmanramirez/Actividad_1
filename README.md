# Actividad 1 - materia: Programación Web

Este repositorio contiene los ejercicios prácticos realizados en la Actividad 1 de la materia de Programación Web.

## Objetivo

Integrar los ejercicios prácticos de Programación Web en un repositorio público de GitHub.

## Objetivos específicos

- Crear una página principal que funcione como menú de acceso a los ejercicios.
- Organizar los archivos del proyecto mediante carpetas.
- Utilizar rutas relativas para enlazar correctamente páginas, estilos, scripts e imágenes.
- Aplicar HTML para la estructura de las páginas.
- Utilizar CSS para mejorar el diseño visual de los ejercicios.
- Implementar JavaScript en las prácticas que requieren interacción o validaciones.
- Publicar el proyecto mediante GitHub Pages.
  
## Contenido del repositorio

El repositorio está organizado de la siguiente manera:

- `index.html`: página principal que funciona como menú de navegación.
- `PrograWeb`: carpeta que contiene una parte de los ejercicios prácticos.
- `ProgramacionWeb`: carpeta que contiene otra parte de los ejercicios prácticos.
- Archivos HTML, CSS, JavaScript e imágenes utilizados en los ejercicios.

## Descripción de la actividad

En esta actividad se reunieron los ejercicios desarrollados durante el tema 2 de la materia de Programación Web.  
Se creó una página principal llamada `index.html`, la cual permite acceder a cada ejercicio mediante enlaces.

También se verificó que los archivos CSS, JavaScript e imágenes funcionaran correctamente fuera de localhost, ya que el proyecto fue publicado mediante GitHub Pages.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap
- SweetAlert
- GitHub
- GitHub Pages
